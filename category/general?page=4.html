  • Server failure

    Saturday morning we had to bury our old faithful web server, after what seems to be a hardware failure claimed it to wherever it is machines go when they die.

    Unfortunately, without us noticing, the same failure corrupted the last few daily backups, rendering them unusable. We've had to rollback the site to a full copy from 23.6.. That means that whatever you did on the site after 23.6. is most likely unrecoverable, if it isn't already recovered. This includes new user registrations, forum posts, profile changes, comments, and so forth.
