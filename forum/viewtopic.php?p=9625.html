I don't really like using the PowerAuras graphics much but prefer using the abilities own icons, you can access this by checking the "Use own icon" -box in the activation menu. When using this option you want to obviously down the size of the graphic from the default 75 to around 20-24 and uncheck the "Glow" -box on top to make the icons distinguishable. When setting these options up there is the small hitch that PowerAuras doesn't seem to know what the ability icon actually looks like until it has "seen" it up on you/your target/whatever once, but at least in latest version it gives you a [?] icon which helps you edit it.
Manni

He is using a UI compilation called Elv22, which is based on the famous TukUI. So the SCT settings might either be TukUI standard or modified by Elv. With TukUI it is hard to name addons since he has included everything into a specific folder structure.

If you look for the font: The font is included in the TukUI package and called combat_font.ttf. It can be found in \Interface\Addons\Tukui\media\fonts\
