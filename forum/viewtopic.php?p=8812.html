I think 81-83 or something has a different expertise cap than 80 and 85. There was a blue post about it last week. That might be where the 23 is coming from.

I've never cared about expertise. Threat has almost never been a big problem except maybe once in caster heavy 10 man group. I don't think that will be changing. Some tanks favour it so they can do more dps and think that in some very hard encounter the tank's extra dps might end up being very helpful but if the fight is that hard the tank probably must concentrate on surviving anyways. So I don't want to sacrifice survivability for extra damage or threat if it isn't really needed.

In WotLK raids I tried to keep hit up so taunts wouldn't miss.
