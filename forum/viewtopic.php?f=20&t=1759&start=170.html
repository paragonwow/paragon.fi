deva wrote:
Reyem wrote:Hey Devai,

How do you make Combat Trance appearing in classtimers? I have it enable in the filters but when I attack anything it doesn't appear.

Sorry for my bad english btw


Go to elvui settings by writing /ec -> Class timer -> filters -> trinket
Then go to dummy and shoot it so u have the Combat Trance on your buffs and mouseover that buff. You should see it's spell id (should look like this 107960 or 109719, depending what version of the trinket you have) then you just "add SpellID" and type the numbers in to that box.

Thats how you add trinket, spells, buffs or what ever you wanna see on you class timers. Class timers cant know about the new trinkets in the game, so if there is new items/trinket procs u wanna see in class timers, this what you need to do every time when you want to see new item/trikets procs on your classtimers.


Thanks! It works perfectly :)
