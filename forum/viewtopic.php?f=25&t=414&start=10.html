If you have the diseases dropping off every once in a while, you can opt to gear for a bit more expertise. It might help with your rotation a bit. Other than that, just go whack on a dummy for a while - you're just probably not keeping it as tight as it should be. The diseases last for 21s and you should be able to refresh them every 20s. Simply sub one BS for a pestilence (you probably know this, but a lot of people came in-game all of a sudden to ask me about this).

On the glyph choice, it doesn't really matter that much. The difference is incredibly minor in any case between ghoul and death coil. If you have plenty of druids in a raid with revitalize, go for the DC glyph if you think your ghoul won't die and you can micro it away from stuff that can kill it. If you have problems with ghoul dying sometimes (and perhaps don't have quite that many druids in a raid), go with the ghoul glyph.

Obsidius wrote:One thing I'm a little confused about, though is how exactly would more expertise be helpful in my rotation? I'm already well over the soft-cap as a DPS.

Yeah. Well, then it won't help at all. I didn't see you link your armory in the original post, I figured you might be getting some dodges and whatnot which might cause a small delay if you're using a set rotation, which in turn might make your diseases fall off.

Also, if you're going with GoD (and ToT/cinderglacier procs), it's probably better to skip the reaping spec and 2xT10 and simply use 4xT9. Once you get the four part bonus on T10, then the reaping spec and set bonus becomes the better choice. This held true for me atleast - I checked it with the ilvl 258 T9 gear I had compared to 251 T10. Most likely it'll hold true for you as well.

Without GoD, the 2xT10 (with reaping) should come out on top though.

Deekaye wrote:...how much harder is it to manage DC to keep it below rune capping? Right now the rotation feels really tight and I worry that adding the extra RP might cause issues.

Not really that much. At some points you might have some RP overflow, but you can try and minimize it. With GoD and Reaping, you'll have two extra GCDs for each rune set in the 20s cycle - it shouldn't pose problems. Even if you overflow, you're going to do a bunch more damage when compared to not getting revitalizes and good AMS absorbs. It's all just free damage.
Synti / Synjin / Vorel
