Hey there.
You shouldn't use any specific rotation for enhancement, cause well when you get 5stack maelstorm weapon, you should use lb right away, and after that see what skills have come off cd to use. So in enha you should use your skills more in a priority order that are off cd. According to Elitist Jerks it should go something like this:

1) Maelstrom Weapon x 5 stacks - Lightning Bolt
2) Lightning shield if not active
3) Spirit Wolves
4) Shamanistic Rage
5) Stormstrike if no SS debuff present
6) Fire Elemental
7) Magma Totem if not active
8) Flame Shock
9) Earthshock
10) Stormstrike
11) Lava Lash
12) Fire Nova
13) Magma Totem
14) Lightning Shield

And ofc don't reapply flame shock if dot is ticking. Shamanistic rage becomes important dps-wise when you get 4p t10 so if you don't just use it whenever you want. If there is multiple targets, Magma Totem and Fire Nova moves up in the priority list
