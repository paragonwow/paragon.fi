Frantic wrote:What is the basic shot priority?
How are lock and load procs best handled?


It's more like priority system, anyways:
Keep serpent sting up 100% of the time -> black arrow (unless explo trap needed on AoE) -> explo shot -> if both of ur black arrow and explo shot are on cd and ur focus is about to cap shoot arcane shot. Then u shoot cobra shot every time ur explo and black arrow are on cd.

Lock and Load:

Normally this is what i do -> explo -> cobra -> explo -> cobra -> explo

If high focus when get Lock and Load proc -> explo -> cobra -> explo -> arcane -> explo

If you have ur black arrow coming out of the cd while Lock and Load proc on -> explo -> black arrow -> explo -> cobra -> explo

Be sure to use every proc on explo and not to waste it on arcane!

Frantic wrote:A question that I would like most answered is how are you guys maximizing burst dps on boss pulls? What has worked best for you (starting rotation)?


It variates hugely on bosses, some bosses you can put all ur cd's up right away, some bosses u need to wait for different phase to use cd's. Other bosses have damage modifiers so u should try to save ur cd's on those.

Normal start: sting up, black arrow up, explo,(*rapid fire,blood fury,call of the wild, agi pot), then lotsa cobras.

*if u can use cd's when ever u want
Mess with the best, die like the rest!

Frantic wrote:A question that I would like most answered is how are you guys maximizing burst dps on boss pulls? What has worked best for you (starting rotation)?


There's no point in optimizing for initial burst on pull really, you want to optimize for best sustained DPS over the whole fight. Which means, Sting up, then normal priorities, pretty much. Some of the fights are in the 5-6min range, in those you really want to blow Rapid Fire right away so you can get it again before the fight is over.
