Using Foobar2000 aswell here! Used to prefer winamp, but it started to get really slow and bloated. Foobar is just so much faster & lighter in like...everything.

For example, adding all my songs in one playlist in winamp with reading the metadata from the files at the sametime would take like a 40 minutes. In Foobar...1 minute!

I haven't done any research how the files are loaded in both programs but Foobar does get the song titles right instantly when the songs are loaded to playlist, unlike winamp which at default adds the files fast to playlist but does not seem to take a look at the metadata. It only reads the metadata when you are seeing the files in playlist so titles are not right and you can't use jump to file correctly until you have scrolled over all songs and waited for winamp to read them...

I also love the Foobars super responsive -search. You can use any word that appears in metadata or filename and Foobar will filter the files containing those words, in a blink.

Foobar is very modular player. There are alot of plugins to add features and modifiy how it looks.

I have modded my Foobar quite a bit with plugins, but haven't arsed to try skin anything...yet.


For movies I use Media player classic with FFDshow codec pack.
