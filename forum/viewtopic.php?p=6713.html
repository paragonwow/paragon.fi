If you want to see his spells, procs, etc check paragon's logs on www.worldoflogs.com (Old log: http://www.worldoflogs.com/reports/6nrm7nplheqhs12z/ ).
Select Synti and hover down from "Dashbord" and select "Expression Editor".
In that quary box copy & paste this:
sourceName = "Synti" and ((type=TYPE_AURA and targetName = "Synti" and fullType != SPELL_AURA_REFRESH) or type=TYPE_CAST)


Select "Run" and you'll be able to see all spells cast and buffs/procs gained and used. It has no use at all to ask for a dummy vid as Synti is using Armor Penetration. Dummies already have 1 armor so you won't see a high dps output.
