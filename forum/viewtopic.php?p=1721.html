@dwarfy

There are couple of things I would like to point out in your gear, you are using 10str15sta as blue gem, use Nightmare Tear ( 10 all stats) and you have 27 expertise without SoV glyph so you should try to get some of those expertise gear swapped to get SoV glyph in full use, try to aim 26~ expertise with the glyph, Exo glyph aint that good, you are also using Glyph of Holy Wrath? I dont know if there is some sensible reason behind this but you should swap that to Consecration, Consecration is great glyph which everyone should use (I have described this in my previous posts).

@Jarett

1. Yes, use Consecration glyph over Exorcism, check my older posts to get more clarified information.
2. No, you shouldnt always gem for socket bonuses. If you gain 4+ strength from socket bonus gem it with 10str/10crit otherwise full 20str. You shouldnt use 10str15sta gems, use one Nightmare Tear (10 all stats) to get your blue gem req.
3. Your FCFS priority is fine and you should always hit DS except when Judgement is off cooldown =).
4. Going oom nowadays is pretty usual if you dont have all the manaregen buffs in raid, arcane intellect, blessing of wisdom and so on. Only things to stop this is try to use DP as soon as you can (when you have free global) and if you see you are going to oom hard just stop using consecration and exo for a second or pop mana potion if you havent used haste pot already.
