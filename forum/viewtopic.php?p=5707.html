The visible ones are following:

Columns UI Most important one
Peakmeter Spectrum Visualisation
Waveform seekbar

+ Lyric show panel nowdays

If you download these plugins and extract them to foobar2000/components -folder you can use my layout file to get the foobar look like mine does.

First download the file somewhere to your computer.

In Foobar:
File->Preferences ->Display->Columns UI ->Main tab -> Inside the "FCL importing and exporting" -box ->click Import and select the file. If you get some errors you might miss some plugins needed.

Yep the columns UI layout editor can get pretty confusing. Basicly you cut segments for objects using splitters, at start the area is one big sheet. The splitters and objects are "read" from top to bottom and left to right.

You can see how the splitters function from this screenshot

I enabled showing of the captions for splitters and oriented them properly to show how splitters crop the area.


The track information is inside the "Item details" object, click it and then click the configure button and you can copypaste or modify the the text showing.

You can also enable more minimalistic item detail bar by going to status bar tab and checking "Show status bar", at the bottom you can insert the code for item details you wish to show.
