Guromin

Living Bomb does more damage per cast time than Fireball so if you can cast it and it does explode it's more damage and worth doing. It takes an element of skill to execute because often it can cause you to sacrifice possible stacked ignites, so timing is everything.

You will not notice a huge increase in dps from this, but it is mathematically more effective.
