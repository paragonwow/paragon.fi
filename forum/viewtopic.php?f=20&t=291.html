Although it looks to be a customized version of Aloft in his UI, I would strongly recommend TidyPlates to anyone looking for an improved nameplate addon from the stock UI.

TidyPlates is very clean & memory efficient, with many different skins (I personally recommend ThreatPlates for both tanks & non-tanks). However, there are other skins like CleanPlates that I think you'll find much, much more appealing than the standard Blizzard name plates.

You can find them on wowinterface.com

If you're dead set on mirroring Maeil's name plates, I guarantee you can do that with Aloft, if you're savvy enough with UI modification.
