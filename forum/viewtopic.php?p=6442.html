It doesn't really make a huge difference exactly when you pot, long as you do it. If the fight is likely to be very short prepotting Wild Magic for the first set of cooldowns probably makes sense, in a longer fight you could perhaps delay the first Rapid Fire that 10 seconds and use a haste potion instead. For the second potion I'd always use haste, preferably when you have some good trinket procs up (and no other haste buff up at the same time).
