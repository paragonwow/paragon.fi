Yeah would be nice if you could reach the block cap (meaning you won't receive anymore fullhit) ( = dodge + parry + block + 5%(miss) = 102.4% )

As long as you are ALWAYS pressing a damage button when attacking the bird, never just be meleeing it, every cooldown should be used.
Before the bird spawns, build up 3 holy power on the initiate, save it til the bird is about to spawn, then hit inquisition, then hit divine plea for 3 holy power, avenging wrath wait for the imprinted debuff to be applied to you, then SHoR the bird, doing this should take a few percent off within a second of it spawning (even more if you get a crit SHoR)
