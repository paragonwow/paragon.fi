The cancelaura macro (cancel) removes health buffs for Anub - yes, I still do it manually :P Then there's a (alga) macro made for Algalon, which nowadays just puts the target to focus, and a macro (t1) for using an on-use trinket... which I haven't been using after Ulduar really.

So just your average stuff and some junk there. Personally I've been using BigWigs for a long time, and generally just prefer it to DBM. It also helps that we have usually made our custom timers to BW before, so we're familiar with it. I don't think there are any major differences though, and unless you guild dictates the use of one, just stick to what you prefer. You might also want to check out DXE, which seems rather slick if you bother configuring it.
