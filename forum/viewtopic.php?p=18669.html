Not from Paragon but just a few points:

-The exact number of targets needed for FoK to be worth using is unconfirmed. Icy Veins says 7+, but I would take that lightly.

-When you say using *just* Tempest, I assume you mean building CPs using single target rotation? That probably isn't viable since for CT to be worth using, FoK would probably also be worth using, so FoK/CT is likely the best aoe rotation.

-Similar to the last point, if FoK/CT are worth using, you may not want to bother keeping SnD/Rupture up, since the
energy you use putting them up could have been used better for aoe (though the energy gained from Venomous Wounds might make a difference).

-It might be possible that if there are only a few targets (say 3 or 4), it might be good to use FoK to apply poisons but then continue with single target rotation, but again not enough testing has been done to confirm this.

Overall, as I'm sure you can see, there needs to be a little more testing before we can say for sure what's good/bad. Hope this helped anyway.
